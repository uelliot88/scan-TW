# 台股掃圖 — 安裝說明

## 前置需求

- Python 3.11（建議版本）
- Git
- 網路連線

---

## Windows 安裝步驟

### 1. 下載專案

```bash
git clone https://github.com/uelliot88/scan-TW.git
cd scan-TW
```

### 2. 建立虛擬環境並安裝套件

```bat
python -m venv venv
venv\Scripts\pip install -r requirements.txt
```

### 3. 執行

直接雙擊 `update_and_open.bat`

腳本會自動：
1. 下載最新股票資料（約 10 分鐘）
2. 推送至 GitHub
3. 開啟瀏覽器顯示結果

---

## macOS 安裝步驟

### 1. 下載專案

```bash
git clone https://github.com/uelliot88/scan-TW.git
cd scan-TW
```

### 2. 建立虛擬環境並安裝套件

```bash
python3 -m venv venv
venv/bin/pip install -r requirements.txt
```

### 3. 設定執行權限（首次一次）

```bash
chmod +x update_and_open.sh
```

### 4. 執行

```bash
./update_and_open.sh
```

或在 Finder 中雙擊 `update_and_open.sh`
（若系統阻擋，請至「系統設定 > 隱私權與安全性」允許執行）

腳本會自動：
1. 下載最新股票資料（約 10 分鐘）
2. 推送至 GitHub
3. 開啟瀏覽器顯示結果

---

## 注意事項

- 首次執行需要 GitHub 帳號並具有此 repo 的推送權限，否則步驟 2（推送）會跳過，但本地資料仍正常更新
- 瀏覽器開啟的是 Streamlit Cloud 部署版本，推送後約 1～2 分鐘會自動更新
- 若需在本地端執行網頁，可改用：`venv/bin/streamlit run app.py`（macOS）或 `venv\Scripts\streamlit run app.py`（Windows）
